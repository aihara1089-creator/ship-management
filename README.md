# MOL 船舶管理リスト — 新造船工事予定ダッシュボード

## 概要
新造船の工事予定・受注状態をチームで共有・管理するダッシュボードです。

## 機能
- **CSVインポート** — 船舶管理CSVをドラッグ＆ドロップでインポート
- **ダッシュボード** — KPIカード、ガントチャート、グラフ表示
- **フィルター** — 船種・所有形態・ステータス・受注状態で絞り込み
- **受注状態管理** — 船ごとに「見積提出済み」「受注済み」を設定・保存
- **チーム共有** — サーバーにデータを保存し複数人でリアルタイム共有
- **自動同期** — 30秒ごとに受注状態を自動ポーリング

## セットアップ

```bash
# 依存パッケージをインストール
npm install

# サーバーを起動（ポート3000）
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

## チームで共有する場合

同じネットワーク内のサーバーで起動し、全員が `http://サーバーIP:3000` にアクセスしてください。

- CSVをアップロードすると全員に共有されます
- 受注状態の変更は30秒以内に全員の画面に反映されます
- ヘッダーの「共有モード」バッジをクリックで即時同期も可能
